# NoteKeeper

A full stack note-taking app built with Laravel.

## Features
- Authentication (register, login, logout)
- Full CRUD for notes
- Category filtering (Personal, Work, Ideas)
- Route protection with Middleware
- Ownership rules with Gates & Policies
- Profile management
- Clean UI with Tailwind CSS + daisyUI

## Tech Stack
- Laravel
- MySQL
- Tailwind CSS + daisyUI
- Vite
